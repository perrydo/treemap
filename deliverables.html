Maintenance & Deliverables

Well... There really isn't any maintenance...

The choice to use open source software was in part for maintenance reasons, and to produce the project in a zero funding based environment. By using Leaflet, Mapbox, KoboToolbox and Github, this map will continue without being moderated or concern of funding changes. Github Actions allowed submissions to be autopopulated onto the web map through scheduled updates, however after 60 days on inactivity on the map, the scheduled updates will pause until re-enabled through GitHub Actions.
